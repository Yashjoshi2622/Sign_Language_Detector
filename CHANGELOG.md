# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Initial release of Advanced Sign Language Studio
- Webcam capture and image upload for gesture input
- Hand landmark extraction using MediaPipe
- Live prediction with heuristic and trained models
- Few-shot learning with sample collection
- Synthetic bootstrap dataset generation
- Model training and evaluation with ensemble classification
- Sentence builder and text-to-speech support
- Training dashboard with holdout accuracy and confusion matrix
- GitHub Actions CI workflow
- Issue and pull request templates
- Contributing guidelines and code of conduct

### Changed
- Improved README with badges and structured sections

### Fixed
- Protobuf compatibility issues for MediaPipe
