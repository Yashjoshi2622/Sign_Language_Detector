# Security Policy

If you discover a security vulnerability in this project, please report it responsibly.

## Reporting a Vulnerability

1. Open a new issue titled `SECURITY: <short description>`.
2. Describe the vulnerability and the affected component.
3. If possible, provide steps to reproduce and any mitigation suggestions.

If you need privacy for a security report, contact the repository owner directly through GitHub.

## Supported Versions

This project is maintained on the latest supported Python versions listed in the repository documentation.
